## Purpose of the Prototype

The primary objectives of this prototype are to:

- Visualize interactions between students and the ride-sharing system  
- Illustrate the complete user flow, from registration to ride matching  
- Present a potential graphical user interface for a future web or mobile application  
- Support academic evaluation, project proposals, and formal presentations  

## Implementation Notes

- The current version of the project is implemented as a **C++ console-based application**, focusing on core functionality and system logic  
- This prototype represents a planned **future user interface extension**, translating the console-based system into a graphical format for an improved user experience
